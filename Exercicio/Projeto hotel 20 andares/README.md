
# Projeto: Hotel de 20 andares
Esse projeto consiste em imprimir os números de 1 a 20, representando os andares de um hotel. Devido a uma superstição do dono desse hotel, foi excluiu o andar de número 13.

## 1º desafio: 
### Escreva um código que imprima todos os números exceto o número 13.
~~~
numeroAndar = 0
for i in range (20):
numeroAndar = numeroAndar+1
  if (numeroAndar == 13):
    continue 
  print (numeroAndar)
~~~~

## 2º desafio:
### Escreva mais um código que resolva o mesmo problema, mas dessa vez usando o laço de repetição 'while'.
~~~
numeroAndar = 0
for i in range (20):
  numeroAndar = numeroAndar+1
  if (numeroAndar == 13):
    continue 
  print (numeroAndar)
~~~

## 3º desafio:  
### Imprima em ordem decrescente (20, 19, 18...)
~~~
numeroAndar = 0
for i in range (20):
  numeroAndar = numeroAndar+1
  if (numeroAndar == 13):
    continue 
  print (numeroAndar)
  ~~~

Estes três desafios demonstram diferentes maneiras de atingir o mesmo objetivo, imprimindo os números de 1 a 20 em ordem crescente e decrescente excluindo o número 13, conforme solicitado no projeto.
